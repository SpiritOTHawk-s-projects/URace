# OS
This game optimized for Windows and Linux.

| Components | Minimum | Recommended |
|----------------|:---------:|----------------:|
| GPU | GT 1030 | GTX 1050 |
| CPU | dual core cpu | quad core cpu |
| Ram | 8 GB | 16 GB |
| Video memory | 2 GB | 4 GB |
